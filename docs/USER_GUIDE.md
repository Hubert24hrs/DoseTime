# DoseTime User Guide

## Getting Started
Welcome to DoseTime! This guide will help you manage your medications and stay healthy.

### Adding a Medication
1. Tap the **+** button on the home screen.
2. Enter the medication name (e.g., "Aspirin").
3. Choose the type (Pill, Liquid, Injection, etc.).
4. Set the dosage (e.g., "100mg").
5. Select frequency (Daily, Weekly, As Needed, etc.).
6. Set the schedule times.
7. Tap **Save**.

### Managing Reminders
- **Take a Dose**: Tap "Take" on the home screen card.
- **Snooze**: Tap "Skip" -> "Take Later" to snooze for 10 minutes.
- **Skip**: Tap "Skip" -> "Skip for Today" to record as skipped.

### Viewing History
1. Go to the **History** tab.
2. Toggle between **Calendar** and **List** views using the icon in the top right.
3. Use the filter button to see history for a specific medication.
4. Tap the **Download/Share** icon to export a PDF report for your doctor.

### Managing Contacts
1. Go to **Settings** > **Contacts**.
2. Tap **+** to add a Doctor or Pharmacy.
3. Enter their details. You can call them directly from the app!

### Data Management
- **Backup**: Go to Settings > Advanced Data Management > Export Data to save a JSON file.
- **Restore**: (Feature coming soon)
- **Delete All**: Use the "Danger Zone" in Data Management to reset the app (requires confirmation).
